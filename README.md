# Spots 

Spots is a photo-sharing page that showcases different tourist attractions that could be found across Japan. Users can browse through multiple photos of different iconic and historic locations, heart the spots that they would plan to visit, edit their profile information, and add any future photos of their likings. 

## Technologies & Techniques

The technologies and techniques that were used for this project were using HTML to create the framework of the webpage and CSS for designing the page. A structured file system waas implemented, following the BEM methodology in order to organize all files for easy inspection. To keep the whole page as user-friendly as possible, the designs were kept very minimal but still responsive for easy usage. Utilizing `@media` queries to create a scalable result for desktops, tablets, and mobile.
